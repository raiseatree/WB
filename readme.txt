LandingStrip is a landing-page template. Designed to be straight-forward, easy to edit and quick to set-up for use.

 - Clean typography: Uses 'Open Sans' and 'Merriweather' fonts from Google Web fonts. Font-size/Line-height appropriate ratios. 
 - Responsive layout and tested across web browsers
 - Uses Twitter Bootstrap's Carousel javascript component for screenshots, use cases or testominals.
 - Plug-and-play: LandingStrip's CSS is in a different file to allow easy editing, or for usage along other versions of Bootstrap
 - HTML5: Classes can be used with HTML5 tags (i.e. header, section and article)
